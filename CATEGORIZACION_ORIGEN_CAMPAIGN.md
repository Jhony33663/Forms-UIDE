## 3. Regla de asignación automática

```
1. Leer UTM de la URL (source, medium, campaign, term)
2. Buscar match en ORIGINS (prioridad: source → medium → campaign)
3. Si NO matchea ORIGINS → heurística por texto en campaign:
     - incluye 'meta'          → 'facebook'
     - incluye 'search/marca'  → 'google search'
     - incluye 'google/pmax'   → 'google pmax'
4. Encontró config → asigna: origen + leadChannel + campaign
5. Si NADA matchea → default según valor actual del campo 'origen':
     - 'Google Natural Search' → 881_MF_TRAFICO_GENERAL_ORGANICO_SEO_SEARCH_IT1_2026
     - 'Formulario Web'       → TRAFICO_GENERAL_OTROS_MEDIOS_IT1_2026
     - Cualquier otro origen   → busca en ORIGINS; si no existe → orgánico
```
